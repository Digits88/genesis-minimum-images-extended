== Description ==

Allows for a specialized image for the "header" image above a post or page's content while allowing a different feature image to be used in the grid loop, Facebook, Pinterest, etc.

== Installation ==

1. Upload contents of the directory to /wp-content/plugins/ (or use the automatic installer)
1. Activate the plugin through the 'Plugins' menu in WordPress
1. In any post or page, upload an image in the new Minimum Image metabox.
1. Verify it works!
